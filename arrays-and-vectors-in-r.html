5 Arrays and Vectors in R

5.1 What is a Vector?

In R, a vector is the fundamental data structure — an ordered collection of elements of the same type. Vectors in R are equivalent to arrays in Octave.

5.2 Creating Vectors

Use c() (combine) to create a vector:

# Row-like numeric vector
row_vec <- c(1, 2, 3, 4, 5)
row_vec
## [1] 1 2 3 4 5
# Character vector
names_vec <- c("apple", "banana", "cherry")
names_vec
## [1] "apple"  "banana" "cherry"

R does not distinguish between row and column vectors by default — all c() vectors are one-dimensional. To make a column vector explicitly, use matrix() with one column (covered in the matrix section).

5.3 Sequence Operator :

The colon : creates a sequence of consecutive integers:

x <- 1:5
x
## [1] 1 2 3 4 5
x <- 5:1   # Counting down
x
## [1] 5 4 3 2 1

5.4 The seq() Function

For sequences with custom step sizes, use seq(from, to, by):

seq(1, 10, by = 2)    # [1, 3, 5, 7, 9]
## [1] 1 3 5 7 9
seq(0, 1,  by = 0.25) # [0.00, 0.25, 0.50, 0.75, 1.00]
## [1] 0.00 0.25 0.50 0.75 1.00
seq(1, 10, length.out = 5)  # 5 evenly spaced points
## [1]  1.00  3.25  5.50  7.75 10.00

5.5 Indexing Elements

In R, individual elements are accessed using square brackets [ ], and indexing starts at 1:

v <- c(10, 20, 30, 40, 50)

v[1]     # First element: 10
## [1] 10
v[3]     # Third element: 30
## [1] 30
v[c(1, 3, 5)]   # Elements 1, 3, and 5
## [1] 10 30 50
v[2:4]           # Elements 2 through 4
## [1] 20 30 40

5.6 Modifying and Deleting Elements

v <- c(10, 20, 30, 40, 50)

v[2] <- 99       # Replace second element
v
## [1] 10 99 30 40 50
v <- v[-3]       # Remove third element
v
## [1] 10 99 40 50

5.7 Useful Vector Functions

v <- c(4, 1, 7, 3, 9, 2)

length(v)    # Number of elements
## [1] 6
sum(v)       # Sum of all elements
## [1] 26
prod(v)      # Product of all elements
## [1] 1512
min(v)       # Minimum value
## [1] 1
max(v)       # Maximum value
## [1] 9
mean(v)      # Arithmetic mean
## [1] 4.333333
sort(v)      # Sort in ascending order
## [1] 1 2 3 4 7 9
rev(v)       # Reverse the vector
## [1] 2 9 3 7 1 4
