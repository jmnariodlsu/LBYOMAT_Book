7 Matrix Algebra in R

7.1 Creating a Matrix

Use matrix(data, nrow, ncol, byrow) to create a matrix. By default, R fills the matrix column by column; set byrow = TRUE to fill row by row:

# Fill by column (default)
A <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3)
A
##      [,1] [,2] [,3]
## [1,]    1    3    5
## [2,]    2    4    6
# Fill by row
B <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3, byrow = TRUE)
B
##      [,1] [,2] [,3]
## [1,]    1    2    3
## [2,]    4    5    6

7.2 Matrix Dimensions

M <- matrix(1:12, nrow = 4, ncol = 3)
M
##      [,1] [,2] [,3]
## [1,]    1    5    9
## [2,]    2    6   10
## [3,]    3    7   11
## [4,]    4    8   12
nrow(M)      # Number of rows: 4
## [1] 4
ncol(M)      # Number of columns: 3
## [1] 3
dim(M)       # Dimensions: c(4, 3)
## [1] 4 3

7.3 Special Matrices

# Identity
diag(3)                      # 3x3 identity matrix
##      [,1] [,2] [,3]
## [1,]    1    0    0
## [2,]    0    1    0
## [3,]    0    0    1
# Matrix of zeros
matrix(0, nrow = 3, ncol = 3)
##      [,1] [,2] [,3]
## [1,]    0    0    0
## [2,]    0    0    0
## [3,]    0    0    0
# Matrix of ones
matrix(1, nrow = 2, ncol = 4)
##      [,1] [,2] [,3] [,4]
## [1,]    1    1    1    1
## [2,]    1    1    1    1
# Diagonal matrix from a vector
diag(c(2, 5, 8))
##      [,1] [,2] [,3]
## [1,]    2    0    0
## [2,]    0    5    0
## [3,]    0    0    8

7.4 Indexing and Slicing

Matrix elements are accessed using [row, col]. Leaving a dimension blank selects all rows or columns:

M <- matrix(1:9, nrow = 3, ncol = 3, byrow = TRUE)
M
##      [,1] [,2] [,3]
## [1,]    1    2    3
## [2,]    4    5    6
## [3,]    7    8    9
M[2, 3]        # Element at row 2, column 3
## [1] 6
M[1, ]         # Entire first row
## [1] 1 2 3
M[, 2]         # Entire second column
## [1] 2 5 8
M[1:2, 2:3]   # Submatrix: rows 1-2, columns 2-3
##      [,1] [,2]
## [1,]    2    3
## [2,]    5    6

7.5 Matrix Addition and Subtraction

Matrices must have the same dimensions for addition and subtraction:

A <- matrix(c(1, 2, 3, 4), nrow = 2, byrow = TRUE)
B <- matrix(c(5, 6, 7, 8), nrow = 2, byrow = TRUE)

A + B    # Matrix addition
##      [,1] [,2]
## [1,]    6    8
## [2,]   10   12
A - B    # Matrix subtraction
##      [,1] [,2]
## [1,]   -4   -4
## [2,]   -4   -4

7.6 Scalar Multiplication

A <- matrix(c(1, 2, 3, 4), nrow = 2, byrow = TRUE)

3 * A    # Multiply every element by 3
##      [,1] [,2]
## [1,]    3    6
## [2,]    9   12

7.7 Element-wise Multiplication

Use * for element-wise multiplication:

A <- matrix(c(1, 2, 3, 4), nrow = 2, byrow = TRUE)
B <- matrix(c(5, 6, 7, 8), nrow = 2, byrow = TRUE)

A * B    # Element-wise (NOT matrix) multiplication
##      [,1] [,2]
## [1,]    5   12
## [2,]   21   32

7.8 Matrix Multiplication

Use %*% for true matrix multiplication. Matrices must be conformable: the number of columns in the first matrix must equal the number of rows in the second:

A <- matrix(c(1, 2, 3, 4), nrow = 2, byrow = TRUE)  # 2x2
B <- matrix(c(5, 6, 7, 8), nrow = 2, byrow = TRUE)  # 2x2

A %*% B    # Matrix multiplication AB
##      [,1] [,2]
## [1,]   19   22
## [2,]   43   50
# Note: matrix multiplication is NOT commutative
B %*% A    # BA — generally different from AB
##      [,1] [,2]
## [1,]   23   34
## [2,]   31   46

7.9 Transpose

Use t() to transpose a matrix — rows become columns and columns become rows. Mathematically, the transpose of \(A\) is denoted \(A^T\) or \(A'\):

A <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, byrow = TRUE)
A
##      [,1] [,2] [,3]
## [1,]    1    2    3
## [2,]    4    5    6
t(A)   # Transpose of A
##      [,1] [,2]
## [1,]    1    4
## [2,]    2    5
## [3,]    3    6

7.10 Trace of a Matrix

The trace is the sum of the diagonal elements. Use sum(diag()):

A <- matrix(c(1, 2, 3, 4, 5, 6, 7, 8, 9), nrow = 3, byrow = TRUE)
A
##      [,1] [,2] [,3]
## [1,]    1    2    3
## [2,]    4    5    6
## [3,]    7    8    9
diag(A)        # Extract diagonal elements
## [1] 1 5 9
sum(diag(A))   # Trace of A: 1 + 5 + 9 = 15
## [1] 15

7.11 Determinant

The determinant tells us whether a matrix is invertible. A matrix is invertible if and only if its determinant is non-zero:

A <- matrix(c(2, 1, 5, 3), nrow = 2, byrow = TRUE)
A
##      [,1] [,2]
## [1,]    2    1
## [2,]    5    3
det(A)   # Determinant of A
## [1] 1

7.12 Matrix Inverse

If \(\det(A) \neq 0\), the inverse \(A^{-1}\) exists. Use solve() to compute it:

A <- matrix(c(2, 1, 5, 3), nrow = 2, byrow = TRUE)

det(A)       # Confirm it's invertible (non-zero)
## [1] 1
solve(A)     # Inverse of A
##      [,1] [,2]
## [1,]    3   -1
## [2,]   -5    2
# Verify: A %*% A^{-1} should equal the identity matrix
round(A %*% solve(A), 10)
##      [,1] [,2]
## [1,]    1    0
## [2,]    0    1

7.13 Solving a System of Linear Equations

solve(A, b) solves the system \(A\mathbf{x} = \mathbf{b}\) for \(\mathbf{x}\). This is more numerically stable than computing solve(A) %*% b explicitly:

# System:  2x1 + x2  = 8
#          5x1 + 3x2 = 19

A <- matrix(c(2, 1,
              5, 3), nrow = 2, byrow = TRUE)

b <- c(8, 19)

x_star <- solve(A, b)
cat("x1 =", x_star[1], "\n")
## x1 = 5
cat("x2 =", x_star[2], "\n")
## x2 = -2
# Verify the solution
A %*% x_star   # Should return c(8, 19)
##      [,1]
## [1,]    8
## [2,]   19

7.14 Eigenvalues and Eigenvectors

A <- matrix(c(4, 1, 2, 3), nrow = 2, byrow = TRUE)

eig <- eigen(A)
eig$values     # Eigenvalues
## [1] 5 2
eig$vectors    # Eigenvectors (columns)
##           [,1]       [,2]
## [1,] 0.7071068 -0.4472136
## [2,] 0.7071068  0.8944272
