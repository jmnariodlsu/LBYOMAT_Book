4 Naming Variables in R

4.1 What is a Variable?

A variable is a named storage location in memory that holds a value. In R, you assign a value to a variable using the assignment operator <- (or =). Once stored, you can reuse that variable anywhere in your code.

For example, we can store the value \(0.75\pi\) as a variable a and then use it in an expression:

a <- 0.75 * pi
a
## [1] 2.356194
a^2 + 10 * sqrt(a)
## [1] 20.90155

When you type an expression without assigning it to a variable, R simply prints the result to the console. You can also capture that result explicitly:

result <- a^2 + 10 * sqrt(a)
result
## [1] 20.90155

4.2 Variable Naming Rules

R has naming conventions you must follow to avoid errors:

Rule Details
Cannot start with a number 2x is invalid; x2 is fine
Case-sensitive Price and price are two different variables
No spaces allowed Use underscores _ or camelCase instead
Avoid reserved words Do not use if, for, TRUE, FALSE, NULL, etc.
Underscores and dots allowed my_var, my.var are both valid 
# Valid variable names
price_1   <- 100
gdp_2023  <- 450000
my.rate   <- 0.05

# Case sensitivity example
GDP <- 1000
gdp <- 2000
GDP == gdp   # FALSE — they are different variables
## [1] FALSE

4.3 Built-in Constants

R has several pre-defined constants:

pi       # 3.14159...
## [1] 3.141593
exp(1)   # Euler's number e ≈ 2.71828
## [1] 2.718282
Inf      # Infinity
## [1] Inf
NaN      # Not a Number (e.g., 0/0)
## [1] NaN
NA       # Missing value
## [1] NA

4.4 Built-in Math Functions

R has a rich set of mathematical functions. The function name is followed by the argument(s) in parentheses:

Math Expression R Function Example
\(|x|\) abs(x) abs(-5) → 5
\(\ln x\) log(x) log(exp(1)) → 1
\(\log_{10} x\) log10(x) log10(100) → 2
\(e^x\) exp(x) exp(1) → 2.718
\(\sqrt{x}\) sqrt(x) sqrt(9) → 3
\(x^n\) x^n 2^3 → 8 
abs(-7)
## [1] 7
log(exp(1))   # Natural log of e = 1
## [1] 1
log10(1000)   # Log base 10
## [1] 3
exp(2)
## [1] 7.389056
sqrt(25)
## [1] 5
3 * exp(2)    # Explicit multiplication is always required
## [1] 22.16717

4.5 Removing Variables

Use rm() to remove a variable from your environment:

x <- 42
x
## [1] 42
rm(x)
exists("x")   # FALSE — x has been removed
## [1] FALSE

To remove all variables at once: rm(list = ls())
